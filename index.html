MarkDown 基本语法

标题

标题用#，后面加空格

列表

有序列表

有序列表, 前面需要加 1. , 2. …点后面一定要加空格。

  1. No.1
  2. No.2
  3. No.3
  4. No.4

无序列表

无序列表 需要在列表前面加 -, 或 *.

  • Item 1
  • Item 2
  • Item 3
  • 项目一
  • 项目二
  • 项目三

  • |, -, :之间的多余空格会被忽略，不影响布局。
  • 默认标题栏居中对齐，内容居左对齐。
  • -: 表示内容和标题栏居右对齐，:- 表示内容和标题栏居左对齐，:-: 表示内容和标题栏居中对齐。
  • 内容和 | 之间的多余空格会被忽略，每行第一个 | 和最后一个 | 可以省略，- 的数量至少有一个。

HEXO入门基础

电脑设置

  • 输入英文时, 中文输入法的 标点符号 全/半角需要选择成为半角. 点击 逗号/句号 图标就可以切换, 半角为实心.
  • 本地模拟地址 http://localhost:4000
  • 编辑文章使用 Geany
  • ctrl-s 保存

Geany 操作说明

文件夹说明

  1. 博客文件夹地址 /home/pi/yiheng/
  2. 文章文件地址 /home/pi/yiheng/source/_posts/hexo-guide.md
  3. 相应文章图片地址 home/pi/yiheng/source/_posts/hexo-guide , _posts文件夹下,于文件名称一样的文件夹.
  4. 点击文件夹中,最上面的2个点 .. 文件夹图标, 会进入上一级文件夹.
  5. 双击文件, 可以打开文章文件.

终端操作

  1. 输入pwd, 查看所在目录. 需要确保在下面这个目录中.

HEXO 常用命令

命令 参数 功能 描述
hexo s 启动本地服务 s为server的缩写
hexo g 生成本地新创建的网页 g为generate的缩写
hexo d 将本地的网页上传到github d为deploy的缩写
hexo new xxx 在本地新建名称为 xxx 的文章 xxx.md 文件就会创建在 _posts 目录下

  1. 启动 网页服务 程序 hexo s 或 hexo server, 得到下面的提示后,可以在http://localhost:4000打开查看.

  2. 新建文章 hexo new test 这样就可以新建一个名字为 test.md的博客文章.

  3. 生成静态博客. hexo g, 或 hexo generate 最后一行 INFO 19 files generated in 1.04 s. 出现,就表示生成完毕.

  4. 上传网站 hexo d或hexo deploy, 就可以上传网站. 出现最后一行INFO Deploy done: git, 就表示上传完毕.

我们全家去旅游

我们全家一起去无锡游玩

今天，我们去了无锡的鼋头渚。小猴子
车子从南港出发，过了好一阵子，我们才到了无锡的鼋头渚。
刚进鼋头渚，就看见一串红色的大帘，上面写着：国家级自然风景区这八个大字。进去发现里面也没什么新鲜的东西，只有一些餐厅。但是一会儿，我看到了轮船！我从来没坐过轮船。于是就跟妈妈说了声，妈妈也正由此意。于是我们一家人挤进了人山人海的长长的队伍中。我们排了快有半个小时的队伍，才成功地坐上了轮船。进了轮船，我就直奔上层。过了一会儿，轮船就开动了。我站在船上观望，兴奋地不得了。开了几分钟，我看到了帆船。那种帆船像极了古时候的帆船。我立刻联想到了三国中的草船借箭的故事。又过了几分钟，我们到了太湖仙岛。我发现精髓都在太湖仙岛上。有动物表演、天仙街、天王府等等。
这个国庆节是在是太有趣了！
