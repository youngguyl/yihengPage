HEXO入门基础

电脑设置

  • 输入英文时, 中文输入法的 标点符号 全/半角需要选择成为半角. 点击 逗号/句号 图标就可以切换, 半角为实心.
  • 本地模拟地址 http://localhost:4000
  • 编辑文章使用 Geany
  • ctrl-s 保存

Geany 操作说明

文件夹说明

  1. 博客文件夹地址 /home/pi/yiheng/
  2. 文章文件地址 /home/pi/yiheng/source/_posts/hexo-guide.md
  3. 相应文章图片地址 home/pi/yiheng/source/_posts/hexo-guide , _posts文件夹下,于文件名称一样的文件夹.
  4. 点击文件夹中,最上面的2个点 .. 文件夹图标, 会进入上一级文件夹.
  5. 双击文件, 可以打开文章文件.

终端操作

  1. 输入pwd, 查看所在目录. 需要确保在下面这个目录中.

  2. 启动 网页服务 程序 hexo s 或 hexo server, 得到下面的提示后,可以在http://localhost:4000打开查看.

  3. 新建文章 hexo new test 这样就可以新建一个名字为 test.md的博客文章.

  4. 生成静态博客. hexo g, 或 hexo generate 最后一行 INFO 19 files generated in 1.04 s. 出现,就表示生成完毕.

  5. 上传网站 hexo d或hexo deploy, 就可以上传网站. 出现最后一行INFO Deploy done: git, 就表示上传完毕.
